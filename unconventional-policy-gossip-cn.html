非传统金融政策
真的拉动了增长吗？

从一条八卦开始的思考实验
——「能证明」和「已被证明」之间，隔着一道深渊
2026年4月29日 Claude & Claude Code / 麦大酱
某处听闻：「本人参考相关研究，理论上证明了日本的收益率曲线控制（YCC）
和负利率等非传统政策对实际增长率产生了正面贡献」

——真的吗？我决定自己先「证明」这个命题，再亲手「摧毁」它。

第一幕：先把命题「证明」了

三条传导路径

路径一：实际利率渠道——名义利率触及零下限（ZLB）后，央行承诺未来制造通胀 → 实际利率（名义利率 - 通胀预期）被压入负值 → 刺激投资和消费 → GDP增长。理论基础：克鲁格曼 (1998)、埃格特松&伍德福德 (2003)。日银的QQE「超调型承诺」正是这个理论的实践版。

路径二：资产组合再平衡——央行大量买入国债 → 长期利率下降（期限溢价被压缩）→ 投资者被迫转向风险资产 → 股价上涨、公司债利差收窄 → 企业融资成本降低 → 设备投资增加 → GDP上升。YCC将这条路径制度化——把10年期利率钉在0%附近。

路径三：汇率渠道——负利率 → 日本与海外利差扩大 → 日元贬值 → 出口企业利润改善 → GDP增长。事实上，2013年QQE启动后日元从80多一路贬到120。

实证研究的背书

论文作者结论
非传统金融政策的效果与副作用开发等（日银, 2024）「对产出和物价有一定的推升效果」
日本非传统金融政策的宏观经济效果宫尾·�的本（RIETI, 2017）「持续提升了实体经济和通胀率」

支持方结论：在理论模型中，通过提升通胀预期、压缩长期利率、推动日元贬值这三条路径，可以在数学上推导出非传统政策对实际增长率的正面影响。部分实证研究也支持这一结论。命题「可以被证明」。

第二幕：亲手摧毁自己的「证明」

摧毁一：前提在现实中没有成立

克鲁格曼模型的核心前提是「通胀预期能被政策推动」。然而QQE启动后近10年，日本的BEI（盈亏平衡通胀率）长期停留在1%上下，远未达到日银期望的2%。前提不成立，证明就是空转。

资产组合再平衡也是同样的困境。银行把国债卖给日银拿到的钱，并没有流向风险资产，而是堆在日银的活期账户里「晒太阳」（日语叫「ブタ積み」——猪存钱）。

摧毁二：「有效」是有条件的

宫尾·冲本 (2017) 发现效果仅在「政策积极期」存在——不是随时有效。RIETI另一项研究 (2022) 发现YCC导入后效果「大幅下降」。日银WP (2024) 的措辞是「一定程度的推升效果」——注意，并没有说「持续推升了实际增长率」。

摧毁三：副作用的净账算不清

「效果和副作用的平衡，随着时间推移不断恶化」
「关于副作用，未知的部分还很多」

—— 木内登英（野村综研，前日银审议委员）

金融机构盈利被压缩、市场功能退化、走向事实上的财政融资——扣除这些副作用后的净效果，没有任何研究给出过「证明」。

摧毁四：反事实不可验证

所有实证研究都依赖于一个「反事实」：如果不实施这些政策，GDP会是多少？但反事实永远不可观测。换一组模型假设，「效果」的数字就变了。这不是「证明」，这是「在特定假设下的推算」。

摧毁五：11年的成绩单

全世界最大规模、最长时间实施非传统金融政策的国家，交出了什么成绩单？

QQE启动（2013）→ 负利率（2016）→ YCC（2016）→ 全部解除（2024）
这11年日本的实际GDP平均增长率：约0.7%

同期　美国：约2.3%　/　欧元区：约1.5%

做了全世界最激进的货币实验的国家，增长率在主要经济体中垫底。
这是现实对「有正面贡献」这个命题最残酷的回答。

结论：凝视深渊

含义与现实的关系
模型上的证明前提A → 结论B 的逻辑推导前提A在现实中是否成立，另当别论
实证上的证明用数据验证因果关系反事实不可观测，严格来说不可能
历史上的证明GDP实际上涨了吗0.7%。很难说涨了

如果「理论上证明了」说的是模型推导，那这是合法的学术活动。但如果把它等同于「已经证明非传统政策对日本增长有贡献」，那就是逻辑的跳跃。

11年，增长率0.7%。

要在这个数字面前说「有正面贡献」，
你需要相信一件无法观测的事：
「如果不这么做，会更糟」。

那是信仰，不是证明。
