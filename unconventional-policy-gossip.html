非伝統的金融政策は
実質成長率にプラスか？

噂から始まる思考実験
——「証明できた」と「証明された」の間に横たわる深淵
2026年4月29日 Claude & Claude Code / 麦大酱
某所にて：「日本の長短金利操作やマイナス金利といった非伝統的政策が
実質成長率にプラスとなることを理論的に証明することができた」

——本当に？ この命題を自分で「証明」し、自分で「破壊」してみる。

第一部：命題を「証明」する

3つの波及経路

経路①　実質金利チャネル：ゼロ金利下限でインフレ期待を引き上げる → 実質金利（名目金利−期待インフレ）がマイナスに → 投資・消費を刺激 → GDP押し上げ。クルーグマン (1998)、エガートソン＆ウッドフォード (2003) が理論的基盤。QQEの「オーバーシュート型コミットメント」はこの実践版。

経路②　ポートフォリオ・リバランス：国債大量買入れ → 長期金利低下 → 投資家がリスク資産へシフト → 株価上昇・社債スプレッド縮小 → 企業の資金調達コスト低下 → 設備投資 → GDP。YCCはこれを制度化。

経路③　為替チャネル：マイナス金利 → 内外金利差拡大 → 円安 → 輸出企業の収益改善 → GDP。実際にドル円は80円台→120円台へ。

実証研究の支持

論文著者結論
非伝統的金融政策の効果と副作用開発ほか（日銀, 2024）「生産・物価に一定の押し上げ効果」
日本の非伝統的金融政策のマクロ経済効果宮尾・沖本（RIETI, 2017）「実体経済とインフレを持続的に向上」

支持側の結論：理論モデル上、ZLB下で期待インフレ引き上げ・長期金利圧縮・為替減価の3経路を通じて、実質成長率にプラスの影響を与えることは数学的に導出できる。実証の一部も支持する。命題は「証明」可能。

第二部：自分の「証明」を破壊する

破壊①　前提が現実で成立しなかった

クルーグマン型モデルは「期待インフレ率が政策で動く」ことを前提とする。しかしQQE導入から10年、BEI（ブレークイーブン・インフレ率）は1%前後に停滞した。前提が成立しなければ、証明は空転する。

ポートフォリオ・リバランスも同様。銀行が国債を日銀に売って得た資金は、リスク資産ではなく日銀当座預金に「ブタ積み」された。

破壊②　「効果あり」は条件付き

宮尾・沖本 (2017) の効果は「積極的な政策姿勢のとき」限定。RIETIの別の研究 (2022) では、YCC導入後は効果が「大きく低下」。日銀WPの「一定の押し上げ効果」も、「実質成長率を持続的に押し上げた」とまでは言っていない。

破壊③　副作用のネット勘定は不明

「効果と副作用のバランスは、時間の経過とともに悪化していった」
「副作用については未知の部分が多い」

—— 木内登英（野村総研、元日銀審議委員）

金融機関収益の圧縮、市場機能の低下、財政ファイナンスへの接近——副作用を差し引いたネットの効果を「証明」した研究は存在しない。

破壊④　反事実は検証不能

全ての実証研究は「やらなかった場合のGDP」という反事実に依存する。しかし反事実は観測できない。モデルの仮定を変えれば「効果」も変わる。これは「証明」ではなく「推計」だ。

破壊⑤　11年のファクトが語ること

非伝統的金融政策を世界で最も大規模かつ長期に実施した国のスコアカード：

QQE開始（2013）→ マイナス金利（2016）→ YCC（2016）→ 解除（2024）
この11年間の日本の実質GDP成長率の平均：約0.7%

同期間　米国：約2.3%　/　ユーロ圏：約1.5%

世界で最も大胆な金融実験を行った国が、主要国中最低の成長率に留まった。
これは「プラスだった」と主張する人への、現実からの最も厳しい反論だ。

結論：深淵を覗く

意味現実との関係
モデル上の証明仮定A → 結論Bの論理的導出仮定Aが現実で成立するかは別問題
実証的な証明データで因果関係を検証反事実が観測不能なため厳密には不可能
歴史的な証明実際にGDPが上がったか0.7%。上がったとは言い難い

「理論的に証明できた」がモデル上の話なら、学術的に妥当な営みだ。しかしそれを「日本の成長に貢献した」という現実の主張に直結させるのは、論理の飛躍である。

11年間で成長率0.7%。

この現実を前にして「プラスだった」と言うためには、
「やらなければもっと悪かった」という反事実を信じる必要がある。

それは信仰であって、証明ではない。
